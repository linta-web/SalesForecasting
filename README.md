# Sales Forecasting using Machine Learning

##  Objective
Predict future sales based on historical retail sales data.

## Tools Used
- Python
- Pandas
- Scikit-learn
- Matplotlib

## Dataset
Retail Sales Dataset

##  Models Used
- Linear Regression
- Random Forest Regressor

##  Result
Predicted future sales trends based on past data.

##  Outcome
This project demonstrates time series forecasting using machine learning models.
